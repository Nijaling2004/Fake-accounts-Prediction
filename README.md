***** Fake Instagram/Facebook Account Prediction using Machine Learning*******

** Project Overview**

This project aims to detect whether an Instagram or Facebook account is **Real** or **Fake** using Machine Learning techniques. It analyzes various account features and predicts the authenticity of a social media profile.

The model is trained on a labeled dataset containing real and fake account information to improve prediction accuracy.


 **Objectives**

- Detect fake social media accounts.
- Improve online security by identifying suspicious profiles.
- Reduce spam, scams, and fake users.
- Build a Machine Learning model with high prediction accuracy.

 **Dataset**

The dataset contains various profile-related features such as:

- Number of Followers
- Number of Following
- Number of Posts
- Profile Picture Availability
- Biography Length
- Username Characteristics
- Account Verification Status
- Engagement Information
- Account Label (Real/Fake)
